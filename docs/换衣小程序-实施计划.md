## 实施检查清单

### 阶段一：后端（FastAPI）

1. 创建 `backend/` 目录及基础结构（app、routers、services、schemas）
2. 创建 `requirements.txt`（fastapi、uvicorn、python-multipart、httpx、python-dotenv）
3. 创建 `.env.example`（DASHSCOPE_API_KEY 占位）
4. 实现 `app/config.py`（读取环境变量、上传限制 5KB-5MB、格式校验）
5. 实现 `app/services/oss_upload.py`（百炼 getPolicy + 上传至临时 OSS，model=aitryon）
6. 实现 `app/services/bailian.py`（创建试衣任务、查询任务结果，Header 含 X-DashScope-OssResourceResolve）
7. 实现 `app/schemas/models.py`（UploadResponse、CreateTryonRequest、CreateTryonResponse、TryonResultResponse）
8. 实现 `app/routers/upload.py`（POST /api/upload/image，接收 multipart，返回 oss_url）
9. 实现 `app/routers/tryon.py`（POST /api/tryon/create、GET /api/tryon/result）
10. 实现 `app/main.py`（挂载路由、CORS 配置）
11. 添加 `backend/README.md`（启动说明、环境变量、API 文档）

### 阶段二：小程序（uni-app）

12. 使用 Vue CLI / HBuilderX 创建 uni-app 项目至 `miniprogram/`，配置 Vue 3
13. 配置 `manifest.json` 支持微信、抖音多端编译
14. 创建首页 `pages/index`：模特选择区、服饰选择区、试衣按钮
15. 实现 `components/model-picker`：预设模特列表 + 上传模特（uni.chooseImage）
16. 实现 `components/garment-picker`：服饰上传（uni.chooseImage）
17. 实现 `api/` 封装：uploadImage、createTryon、getTryonResult
18. 创建结果页 `pages/result`：展示试衣图、下载/分享
19. 实现完整试衣流程：选择 → 上传 → 创建任务 → 轮询结果 → 跳转展示
20. 添加 3 张以上预设模特图至 `static/preset-models/`（占位图或 CDN 配置说明）

### 阶段三：联调与文档

21. 更新项目根目录 `README.md`：项目说明、backend 与 miniprogram 启动方式、配置指引
